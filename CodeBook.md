The features in the presented datasets comes from accelerometer and gyroscope data capture, the original data
is in https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip.

All the variables preserve the original precision and type defined in feature_info.txt file in source files,
the subject and activity feature were added to complete the required info for analysis.  subject feature contains
unique id assigned to each one of 30 volunteers in the experiment[1-30].

Activity feature describe the activity performed by the volunteer while the accelerometer and gyroscope capture info, the values
are WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING
